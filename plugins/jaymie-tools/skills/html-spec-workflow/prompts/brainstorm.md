# Prompt 模板：HTML 头脑风暴

> 复制下方文本到 Claude，把方括号里的内容替换成你的实际场景。

---

## 标准版

```
我现在在思考 [题目 / 上下文 / 想要解决的问题]。

请你不要给我文字列表——直接把 5–8 个想法写成一份 **HTML 文件**。

每个想法包含：
- 标题（中文）
- 1–3 句中文说明
- 一个用 div + CSS 表达的可视化小 mockup（不要插图片）
- 风险评估卡（高 / 中 / 低 + 一句风险描述）
- "为什么值得做"的小注解

整页中文，简洁现代风格。零外部依赖（系统字体即可）。
我相信你的判断，按你认为最有吸引力的方式排版。

输出到 ./outputs/brainstorm-[简短英文 slug].html
```

---

## 快速版（场景已经很清楚时）

```
帮我针对 [题目] 用 HTML 文件做头脑风暴，5–8 个想法，每个带小 mockup、
中文说明、风险评估。整页中文。Hey Claude, I trust you here.
存到 ./outputs/brainstorm-[slug].html
```

---

## 加约束版（已经有偏好方向时）

```
针对 [题目] 做 HTML 头脑风暴。

约束：
- 至少 [N] 个想法
- 必须包含一个 [类型] 方向的方案（保底）
- 必须包含一个"看起来很疯但可能很好"的方案
- 技术栈倾向：[xxx]

但其余你自由发挥。中文。存到 ./outputs/brainstorm-[slug].html。
```
